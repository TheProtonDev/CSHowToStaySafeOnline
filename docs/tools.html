Available Tools To Help You With Security:

Password Manager

In order to stay safe online you need secure passwords that cannot be cracked in a matter of seconds or minutes, to do this efficiently you'll want to use a password manager, luckily for you you have plenty of options.

Bitwarden | Get Here

  • Free
  • Open Source
  • MultiPlatform Program Available
  • Browser Extension

LastPass | Get Here

  • Freemium
  • Open Source
  • Nice user interface
  • Browser extension

Virus Protection

Another important step to take in living a safe digital life is using Virus Protection tools to make sure your PC is safe, these aren't a requirement if you are extremely safe about browsing technically, but are suggested for the average user

MalwareBytes | Get Here

  • Freemium
  • Reputable
  • Free Basic Features Such As, Scanning, Quarantining, and more

Bullguard | Get Here

  • Paid / Trial Available
  • Good/Intuitive UI
  • Lots of features such as safe browsing, game boosting, cloud backup, home network scanner, identity protection, and parental controls
  • - Limited support on Mac and no support for IOS
