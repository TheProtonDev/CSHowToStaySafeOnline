Security Tips

To stay safe online you need to make sure you set some ground rules down for yourself to stay safe

Questions to Determine if You're at Risk

  • Have you opened any suspicious emails?
  • Have you responded to any suspicious emails?
  • Have any of your accounts been breached?
  • Do you have any personal information available to the public and/or coworkers within the company?

Password Tips

  • Use complex passwords i.e symbols, capitals, numbers, two-words, etc
  • Use different passwords for different things
  • Use a password manager to generate passwords to only have to remember one secure password, some are linked on the tools page
  • Use the howsecureismypassword tool linked on the tools page when creating a password to see how long it would take to crack
